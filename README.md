## 📋 C++ STL Cheat Sheet (Used in Solutions)

| Container | Use Case |
|-----------|----------|
| `vector<int>` | Dynamic arrays |
| `unordered_map<int,int>` | Hash maps for O(1) lookup |
| `unordered_set<int>` | Hash sets for uniqueness check |
| `stack<int>` | LIFO problems |
| `queue<int>` | BFS, FIFO problems |
| `priority_queue<int>` | Heap / Top-K problems |
| `sort()` | Sorting arrays |
| `lower_bound()` / `upper_bound()` | Binary search |
